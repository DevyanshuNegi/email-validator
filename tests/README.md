# Tests\n\nTest files, test data, and test results.\n\n## Structure\n\n- `data/`: Input test data (CSV files with email lists)\n- `results/`: Generated test results (CSV, JSON, summaries)\n- `test-performance.py`: Python performance testing script\n- `requirements-test.txt`: Python dependencies\n\n## Usage\n\n```bash\n# Install dependencies\npip install -r tests/requirements-test.txt\n\n# Run performance tests\ncd tests\npython3 test-performance.py data/test-emails-small.csv\n```\n\n## Test Data\n\n- `test-emails.csv`: Full test dataset\n- `test-emails-small.csv`: Small test dataset (5 emails)\n\n## Test Results\n\nResults are automatically saved to `results/` with timestamps:\n- `test-results-YYYYMMDD_HHMMSS.csv`\n- `test-results-YYYYMMDD_HHMMSS.json`\n- `test-summary-YYYYMMDD_HHMMSS.txt`
